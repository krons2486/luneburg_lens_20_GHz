# luneburg_lens_20_GHz
Модель антенны - перфорированной сферической линзы Люнебурга с вертикальной поляризацией на рабочей частоте 20 GHz диаметром 108 мм (7.2λ), облучаемая E-секторальным рупорным облучателем 6.8x8.8 мм. Материал линзы: относительная диэлектрическая проницаемость ε=2.56. Материал оболочки ε=1.069156. Расчёт проведён в программе Ansys HFSS.

КУ: 26 дБ; КИП: 0.778; ширина главного лепестка в E плоскости 7.42 градуса, в H плоскости 8.72 градуса; уровень боковых лепестков -15.2 дБ в E плоскости, -20.2 дБ в H плоскости; F/B: -33.1 дБ
